## About Bandit
Bandit is a wargame designed to teach the basics of Linux command line through progressively challenging levels. Each level requires solving a puzzle to get the password for the next level.

## Skills Learned by Level

### Levels 0-5: Basics
- SSH connections
- File navigation (ls, cd, cat)
- Hidden files
- File types
- Special characters in filenames

### Levels 6-10: Searching
- find command
- grep command
- File properties
- Text processing

### Levels 11-15: Encoding & Data
- Base64 encoding
- ROT13 cipher
- Hex dumps
- Compression (tar, gzip)

### Levels 16-20: Advanced
- Network connections
- SSL/TLS
- Port scanning
- Process management
- Scripting
